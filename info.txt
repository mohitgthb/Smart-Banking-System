    #interface:
        "implements" is used when a class wants to use an interface
        It means the class promises to provide implementation of interface methods

        interface Animal {
          void sound();   // method declaration
        }

        class Dog implements Animal {
            public void sound() {
                System.out.println("Bark");
            }
        }
